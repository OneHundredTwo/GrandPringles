# MasterOfMountain

> 대한민국 100대 명산 데이터를 바탕으로 등산로 및 등산커뮤니티기능을 제공하는 안드로이드 어플리케이션 입니다.


## [팀 소개 및 기획]()

## 개발 환경
* **개발언어** : JAVA 9, XML, JSP(el, jstl)
* **WAS** : Apache Tomcat 9
* **Server Framework** : Spring 5, Spring web MVC 5(RESTful), Mybatis 3.4
* **IDE** : Eclipse 4.6 Neon, Android Studio 2.3 
* **OS** : windows 10
* **DBMS** : Mysql 5.7
